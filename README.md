# Flask Leave Management Demo

A simple Flask-based web application to manage leave requests. This project demonstrates basic CRUD operations, form handling, and routing using the Flask framework.

---

## 🚀 Features

- Submit leave requests
- View all leave applications
- Approve / Reject leave (if implemented)
- Simple and clean UI
- Lightweight and easy to run

---

## 🛠️ Tech Stack

- Python
- Flask
- HTML / CSS (for frontend)
- SQLite (or any database you used)
